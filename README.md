# arduino-distance-reader
The following circuit and code is for an Arduino distance reader using an ultrasonic sensor.
This measures distance by calculating the duration of the pulse of an ultrasonic wave from the sensor (microseconds) multiplied by the speed of sound in cm/Î¼s and dividing the result by 2 to get the distance from an object to the sensor.
